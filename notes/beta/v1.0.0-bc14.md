#### misc: add release and changelog make targets [#40](https://github.com/elastic/cloud-sdk-go/pull/40)

push releases to the github repository.

### Bug fixes

#### api: Add CloudClientRuntime client.Runtime wrapper [#46](https://github.com/elastic/cloud-sdk-go/pull/46)

Adds a new structure which implements runtime/client.ClientTransport
and allows predefined rules to call a regionless or region prefixed URL
API client.

This change is necessary since the /deployments path does not work
with the region interpolated in the URL.
